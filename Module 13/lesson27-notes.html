Working with Web-Based Forms

This lesson taught us all about how to set up forms on the front end, which are what we see all the time when navigating the web. First, we learned how to create a form using the "form" tag and specifying its attributes. These details set the foundation for where your data will go and what this will look like after the form is submitted. Then, we learned about the "input" tag, which specifies the type of form the user will fill out. The text input, for example, will allow you to insert text in the form field, and you can specify various attributes to customize the field, such as adding placeholder text. Similarly, there are checkboxes, radio buttons, and selection lists that you can choose from.

More definitions

Conditional Statements:

A statement in JavaScript that consists of a condition and an action. It allows your scripts to behave differently based on the values or variables inputted.

Using Multiple Conditional Statements:

Here's an example of using multiple conditions with if and else statements:

if (hour_of_day < 10){ document.write("Good morning.");
}else if ((hour_of_day >= 14) && (hour_of_day <= 17)){
document.write("Good afternoon.");
}else if (hour_of_day >= 17){
document.write("Good evening.");
}else {
document.write("Good day.");
}

Conditional Operators:

Here are some possible conditional operators:

  • ==—Is equal to
  • !=—Is not equal to
  • <—Is less than
  • >—Is greater than
  • >=—Is greater than or equal to
  • <=—Is less than or equal to

Loops:

You can use loops to keep track of how many times a statement runs and execute a function when the specified limit is reached. For example, use loops to search for values in a large data set repetitively.

Event Handlers:

Event handlers are scripts that you create and use to detect and respond to an event.

"Click" Handlers:

To detect a click, you can use mouseup and mousedown properties for the “click” handler.

Keystrokes:

To make your page listen for keystrokes, use the keypress event handler. Use the event.which property and the from-CharCode string method to store the value of the key and tell you which key was pressed.


For example:

function displayKey(e) {
// which key was pressed?
var keycode=e.which;
character = String.fromCharCode(keycode); // find the object for the destination paragraph
var keys_paragraph = document.getElementById('keys');
// add the character to the paragraph
keys_paragraph.innerHTML += character;
}

The window object:

The window object has the following properties:

  • window.closed
  • window.name
  • window.opener
  • window.outerHeight and window.outerWidth
  • window.screen
  • window.self
